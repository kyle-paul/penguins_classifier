# Penguins Classifier
## Description:
This Streamlit app can predict to which type the penguin with user's input features might belong: **"Adelie", "Chinstrap", "Gentoo"** (3 types of penguins)

## Requirements:
  - Streamlit
  - Pickle
  - Numpy
  - Pandas
  - Matplotlib
  - Tensorflow keras
  - Sklearn
  - Plotly
